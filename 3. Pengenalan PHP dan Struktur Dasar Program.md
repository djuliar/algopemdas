> **PHP** adalah singkatan dari **PHP: Hypertext Preprocessor**
> PHP adalah **bahasa pemrograman yang digunakan untuk membuat website dinamis**.

Artinya:

* Website bisa **berinteraksi dengan pengguna** (bukan hanya tampilan statis).
* Bisa **mengolah data**, **menampilkan informasi dari database**, dan **mengubah isi halaman secara otomatis**.

## ğŸ’¡ Contoh Website Dinamis

| Website         | Contoh Fungsinya                           | PHP di Baliknya                     |
| --------------- | ------------------------------------------ | ----------------------------------- |
| Website Sekolah | Menampilkan daftar siswa dari database     | PHP menampilkan data otomatis       |
| Toko Online     | Menampilkan produk, menghitung total harga | PHP memproses dan menampilkan hasil |
| Formulir Kontak | Menerima pesan dari pengguna               | PHP menyimpan data ke server        |

Jadi, **PHP bekerja di sisi server** (server-side), sedangkan **HTML, CSS, dan JavaScript** bekerja di sisi tampilan (client-side).
