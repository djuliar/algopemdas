### ğŸ§© **1. Pengantar WordPress**

#### a. Apa itu WordPress?

WordPress adalah **Content Management System (CMS)** berbasis open-source yang digunakan untuk membuat dan mengelola website secara mudah tanpa harus menulis kode dari nol.
Dikembangkan menggunakan bahasa **PHP** dan **database MySQL/MariaDB**.

#### b. Jenis WordPress:

1. **WordPress.com**

   * Platform hosting siap pakai (dikelola oleh Automattic).
   * Cocok untuk pemula tanpa keahlian teknis.
   * Terbatas dalam kustomisasi (fitur berbayar untuk kustom lebih lanjut).

2. **WordPress.org**

   * Self-hosted (dapat diinstal di server sendiri).
   * Memberikan kebebasan penuh untuk mengatur tema, plugin, dan konfigurasi.
   * Cocok untuk profesional atau developer.

### âš™ï¸ **3. Instalasi WordPress**

#### a. Persyaratan Sistem:

* PHP versi 7.4 ke atas
* MySQL 5.7 / MariaDB 10.3 ke atas
* Apache atau Nginx Web Server
* 512 MB RAM minimal (disarankan 1 GB ke atas)

#### b. Langkah Instalasi:

1. Unduh WordPress dari [wordpress.org](https://wordpress.org/download/).
2. Ekstrak ke folder server lokal (misalnya `www/wordpress` untuk Laragon).
3. Buat database baru di phpMyAdmin.
4. Jalankan `localhost/wordpress` di browser.
5. Isi detail koneksi database â†’ buat akun admin â†’ instalasi selesai.
