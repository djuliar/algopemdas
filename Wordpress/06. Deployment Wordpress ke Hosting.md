# **Tahapan Lengkap Deployment WordPress ke Hosting**
Kita akan memindahkan WordPress dari **Laragon (lokal)** ke **hosting dengan subdomain `dispenduk.workshop-trpl.com`** sebagai contoh, menggunakan **WinSCP** untuk upload file .sql ke **PhpMyAdmin hosting** untuk impor database.

---

## üß© 1. Persiapan di Lokal (Laragon)

Pastikan WordPress kamu sudah berjalan normal di Laragon.

### a. Cek Folder Proyek

Biasanya berada di:

```
C:\laragon\www\nama_proyek_wordpress
```

Di dalamnya ada file seperti:

```
wp-admin/
wp-content/
wp-includes/
wp-config.php
index.php
...
```

### b. Export Database dari Laragon

1. Buka `http://localhost/phpmyadmin/`
2. Pilih database WordPress kamu.
3. Klik **Export ‚Üí Quick ‚Üí SQL ‚Üí Go**
4. Simpan file hasil export, misalnya: `dispenduk.sql`

## ‚öôÔ∏è 4. Konfigurasi `wp-config.php`

Setelah file ter-upload, **edit file `wp-config.php`** di hosting agar konek ke database hosting.

Ubah bagian ini:

```php
/** The name of the database for WordPress */
define( 'DB_NAME', 'u731511898_dispenduk' );

/** MySQL database username */
define( 'DB_USER', 'u731511898_dispenduk' );

/** MySQL database password */
define( 'DB_PASSWORD', 'password_dari_dosen' );

/** MySQL hostname */
define( 'DB_HOST', 'localhost' );
```

Simpan perubahan.

## üß≠ 5. Ubah URL Situs di Database

Setelah diimpor, URL WordPress masih menggunakan alamat lokal (`localhost/dispenduk`), jadi perlu diganti.

### Opsi 1 ‚Äî Melalui PhpMyAdmin

1. Buka tabel `wp_options`
2. Ubah nilai pada:

   * `siteurl` ‚Üí `https://dispenduk.workshop-trpl.com`
   * `home` ‚Üí `https://dispenduk.workshop-trpl.com`

### Opsi 2 ‚Äî Melalui Plugin (jika bisa login)

Install plugin seperti **Better Search Replace** untuk mengganti semua link `localhost/...` menjadi `https://dispenduk.workshop-trpl.com`.

## üîß 6. Cek dan Uji Website

Buka di browser:

```
https://dispenduk.workshop-trpl.com/
```

Jika tampil website WordPress kamu ‚Äî berarti sukses! üéâ

Jika muncul error seperti:

* **Error establishing a database connection** ‚Üí cek kembali `wp-config.php`
* **500 Internal Server Error** ‚Üí cek `.htaccess` dan pastikan formatnya benar:

  ```htaccess
  # BEGIN WordPress
  <IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteRule ^index\.php$ - [L]
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule . /index.php [L]
  </IfModule>
  # END WordPress
  ```
