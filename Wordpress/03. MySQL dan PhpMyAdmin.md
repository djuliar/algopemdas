# MySQL dan phpMyAdmin

## üß© **1. Pengantar MySQL**

### a. Apa itu MySQL

**MySQL** adalah *Relational Database Management System* (RDBMS) berbasis **SQL (Structured Query Language)** yang digunakan untuk mengelola data secara terstruktur.
MySQL bersifat **open source** dan banyak digunakan bersama **PHP** untuk membangun aplikasi web dinamis.

### b. Karakteristik MySQL

* **Open Source & Gratis** ‚Äì Dapat digunakan tanpa biaya.
* **Relational** ‚Äì Data disimpan dalam tabel dan dihubungkan melalui relasi.
* **Portabilitas tinggi** ‚Äì Berjalan di berbagai platform (Windows, Linux, macOS).
* **Kecepatan tinggi & handal** ‚Äì Cocok untuk aplikasi skala kecil hingga besar.
* **Multi-user** ‚Äì Banyak pengguna dapat mengakses database secara bersamaan.
* **Keamanan kuat** ‚Äì Mendukung sistem login dan hak akses (privilege system).

### c. Fungsi MySQL

* Menyimpan data aplikasi secara terstruktur.
* Melakukan operasi CRUD (Create, Read, Update, Delete).
* Mendukung query kompleks dan transaksi.
* Menjadi backend database untuk aplikasi web, desktop, atau mobile.

## üß† **2. Konsep Dasar Database MySQL**

### a. Istilah penting:

| Istilah         | Penjelasan                                      |
| --------------- | ----------------------------------------------- |
| **Database**    | Kumpulan tabel yang menyimpan data terorganisir |
| **Table**       | Struktur data dalam bentuk baris dan kolom      |
| **Field**       | Kolom yang mewakili atribut data                |
| **Record**      | Baris yang berisi data nyata                    |
| **Primary Key** | Kolom unik yang mengidentifikasi setiap record  |
| **Foreign Key** | Kolom yang menjadi penghubung antar tabel       |
| **Query**       | Perintah SQL untuk mengolah data                |

### b. Struktur Data Relasional

MySQL menyimpan data dalam **tabel-tabel** yang dapat berelasi.
Contoh relasi antar tabel:

* **One to One**: Setiap baris di tabel A terhubung ke satu baris di tabel B.
* **One to Many**: Satu baris di tabel A terhubung ke banyak baris di tabel B.
* **Many to Many**: Banyak baris di tabel A berhubungan dengan banyak baris di tabel B (biasanya melalui tabel penghubung).

## üßÆ **3. Dasar Perintah SQL dalam MySQL**

### a. Manajemen Database

```sql
CREATE DATABASE nama_database;
SHOW DATABASES;
USE nama_database;
DROP DATABASE nama_database;
```

### b. Manajemen Tabel

```sql
CREATE TABLE mahasiswa (
    nim INT PRIMARY KEY,
    nama VARCHAR(50),
    jurusan VARCHAR(30)
);

SHOW TABLES;
DESCRIBE mahasiswa;
DROP TABLE mahasiswa;
```

### c. Operasi CRUD (Create, Read, Update, Delete)

```sql
-- INSERT DATA
INSERT INTO mahasiswa (nim, nama, jurusan)
VALUES (12345, 'Budi', 'Informatika');

-- READ DATA
SELECT * FROM mahasiswa;
SELECT nama, jurusan FROM mahasiswa WHERE jurusan='Informatika';

-- UPDATE DATA
UPDATE mahasiswa
SET nama='Budi Santoso'
WHERE nim=12345;

-- DELETE DATA
DELETE FROM mahasiswa WHERE nim=12345;
```

### d. Query Lanjutan

```sql
-- Menyortir data
SELECT * FROM mahasiswa ORDER BY nama ASC;

-- Mencari data unik
SELECT DISTINCT jurusan FROM mahasiswa;

-- Menggabungkan tabel (JOIN)
SELECT mahasiswa.nama, nilai.nilai
FROM mahasiswa
JOIN nilai ON mahasiswa.nim = nilai.nim;

-- Fungsi agregat
SELECT COUNT(*), AVG(nilai) FROM nilai;
```

## üß∞ **4. Pengantar phpMyAdmin**

### a. Apa itu phpMyAdmin

**phpMyAdmin** adalah aplikasi berbasis web yang digunakan untuk mengelola **MySQL** melalui antarmuka grafis (GUI).
Aplikasi ini biasanya berjalan bersama **Laragon, XAMPP, LAMPP, atau WAMP**.

### b. Fitur Utama phpMyAdmin

* Membuat, mengedit, dan menghapus database & tabel.
* Menjalankan perintah SQL tanpa terminal.
* Mengimpor & mengekspor database (format SQL, CSV, XML, dll).
* Mengatur hak akses user.
* Melihat relasi antar tabel (diagram ERD).
* Backup & restore database dengan mudah.

### c. Cara Mengakses phpMyAdmin

1. Jalankan **Laragon Control Panel** ‚Üí Start **Apache** dan **MySQL** atau tekan tombol Start All.
2. Buka browser ‚Üí ketik `http://localhost/phpmyadmin`.
3. Anda akan masuk ke dashboard phpMyAdmin.

### d. Jika phpMyAdmin tidak bisa diakses

Biasanya PhpMyAdmin tidak bisa dibuka karena salah satu hal berikut:

1. Folder **phpMyAdmin** belum ada di direktori `C:\laragon\etc\apps`.
2. Konfigurasi Apache di Laragon **belum mengizinkan akses** ke folder tersebut.
3. Port Apache sudah dipakai aplikasi lain.
4. File konfigurasi **phpMyAdmin.conf** salah atau belum disertakan.
5. Permission folder tidak sesuai (error *Forbidden* atau *Access Denied*).

### e. Cek Apakah PhpMyAdmin Sudah Terpasang di Laragon

1. Buka folder:

   ```
   C:\laragon\etc\apps
   ```
2. Lihat apakah ada folder `phpMyAdmin-<versi>`
   Contoh: `phpMyAdmin-5.2.1`
3. Jika **tidak ada**, unduh versi terbaru dari situs resmi:
   üëâ [https://www.phpmyadmin.net/downloads/](https://www.phpmyadmin.net/downloads/)
   lalu **ekstrak ke folder apps** rename menjadi **phpMyAdmin**:

   ```
   C:\laragon\etc\apps\phpMyAdmin
   ```

### f. Jika login `root` gagal dengan pesan password tidak boleh kosong.

`AllowNoPassword` adalah opsi konfigurasi dalam file **`config.inc.php`** milik phpMyAdmin yang **mengizinkan user login tanpa password**, khususnya untuk akun seperti `root` yang memiliki password kosong di MySQL.

Secara default, phpMyAdmin **tidak mengizinkan login tanpa password** demi alasan keamanan.
Namun, untuk kebutuhan *local development* (misalnya di Laragon, XAMPP, WAMP), kita bisa mengaktifkan fitur ini.

---

#### 1. Letak Konfigurasi

Rename file `config.sample.inc.php` menjadi `config.inc.php`. File konfigurasinya berada di:

```
C:\laragon\etc\apps\phpMyAdmin\config.inc.php
```

---

#### 2. Contoh Penggunaan

Tambahkan atau ubah baris berikut di dalam file tersebut:

```php
$cfg['Servers'][$i]['AllowNoPassword'] = true;
```

1. Buka tab **User accounts**.
2. Tambahkan user baru dengan username dan password.

## üß± **7. Hubungan MySQL dan PHP**

MySQL sering digunakan bersama **PHP** dalam pengembangan web:

* PHP berfungsi untuk memproses logika aplikasi.
* MySQL menyimpan data pengguna, produk, transaksi, dll.
* Keduanya berkomunikasi melalui fungsi atau ekstensi seperti:

  * `mysqli_connect()`, `mysqli_query()`
  * atau menggunakan **PDO (PHP Data Objects)**.

Contoh koneksi PHP ke MySQL:

```php
<?php
$koneksi = mysqli_connect("localhost", "root", "", "db_mahasiswa");

if (!$koneksi) {
    die("Koneksi gagal: " . mysqli_connect_error());
}
echo "Koneksi berhasil!";
?>
```

## üíæ **8. Studi Kasus**

### A. Sistem Manajemen Mahasiswa

Tabel:

* `mahasiswa (nim, nama, jurusan)`
* `nilai (id_nilai, nim, mata_kuliah, nilai)`

Relasi: `mahasiswa.nim = nilai.nim`
Contoh Query:

```sql
SELECT mahasiswa.nama, nilai.mata_kuliah, nilai.nilai
FROM mahasiswa
JOIN nilai ON mahasiswa.nim = nilai.nim
WHERE nilai.nilai > 80;
```
