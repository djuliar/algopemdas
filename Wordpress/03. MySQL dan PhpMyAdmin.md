### a. Apa itu MySQL

**MySQL** adalah *Relational Database Management System* (RDBMS) berbasis **SQL (Structured Query Language)** yang digunakan untuk mengelola data secara terstruktur.
MySQL bersifat **open source** dan banyak digunakan bersama **PHP** untuk membangun aplikasi web dinamis.

### b. Karakteristik MySQL

* **Open Source & Gratis** â€“ Dapat digunakan tanpa biaya.
* **Relational** â€“ Data disimpan dalam tabel dan dihubungkan melalui relasi.
* **Portabilitas tinggi** â€“ Berjalan di berbagai platform (Windows, Linux, macOS).
* **Kecepatan tinggi & handal** â€“ Cocok untuk aplikasi skala kecil hingga besar.
* **Multi-user** â€“ Banyak pengguna dapat mengakses database secara bersamaan.
* **Keamanan kuat** â€“ Mendukung sistem login dan hak akses (privilege system).

### c. Fungsi MySQL

* Menyimpan data aplikasi secara terstruktur.
* Melakukan operasi CRUD (Create, Read, Update, Delete).
* Mendukung query kompleks dan transaksi.
* Menjadi backend database untuk aplikasi web, desktop, atau mobile.

## ğŸ§  **2. Konsep Dasar Database MySQL**

### a. Istilah penting:

| Istilah         | Penjelasan                                      |
| --------------- | ----------------------------------------------- |
| **Database**    | Kumpulan tabel yang menyimpan data terorganisir |
| **Table**       | Struktur data dalam bentuk baris dan kolom      |
| **Field**       | Kolom yang mewakili atribut data                |
| **Record**      | Baris yang berisi data nyata                    |
| **Primary Key** | Kolom unik yang mengidentifikasi setiap record  |
| **Foreign Key** | Kolom yang menjadi penghubung antar tabel       |
| **Query**       | Perintah SQL untuk mengolah data                |

### b. Struktur Data Relasional

MySQL menyimpan data dalam **tabel-tabel** yang dapat berelasi.
Contoh relasi antar tabel:

* **One to One**: Setiap baris di tabel A terhubung ke satu baris di tabel B.
* **One to Many**: Satu baris di tabel A terhubung ke banyak baris di tabel B.
* **Many to Many**: Banyak baris di tabel A berhubungan dengan banyak baris di tabel B (biasanya melalui tabel penghubung).

## ğŸ§® **3. Dasar Perintah SQL dalam MySQL**

### a. Manajemen Database

```sql
CREATE DATABASE nama_database;
SHOW DATABASES;
USE nama_database;
DROP DATABASE nama_database;
```

### b. Manajemen Tabel

```sql
CREATE TABLE mahasiswa (
    nim INT PRIMARY KEY,
    nama VARCHAR(50),
    jurusan VARCHAR(30)
);

SHOW TABLES;
DESCRIBE mahasiswa;
DROP TABLE mahasiswa;
```

### c. Operasi CRUD (Create, Read, Update, Delete)

```sql
-- INSERT DATA
INSERT INTO mahasiswa (nim, nama, jurusan)
VALUES (12345, 'Budi', 'Informatika');

-- READ DATA
SELECT * FROM mahasiswa;
SELECT nama, jurusan FROM mahasiswa WHERE jurusan='Informatika';

-- UPDATE DATA
UPDATE mahasiswa
SET nama='Budi Santoso'
WHERE nim=12345;

-- DELETE DATA
DELETE FROM mahasiswa WHERE nim=12345;
```

### d. Query Lanjutan

```sql
-- Menyortir data
SELECT * FROM mahasiswa ORDER BY nama ASC;

-- Mencari data unik
SELECT DISTINCT jurusan FROM mahasiswa;

-- Menggabungkan tabel (JOIN)
SELECT mahasiswa.nama, nilai.nilai
FROM mahasiswa
JOIN nilai ON mahasiswa.nim = nilai.nim;

-- Fungsi agregat
SELECT COUNT(*), AVG(nilai) FROM nilai;
```

## ğŸ§° **4. Pengantar phpMyAdmin**

### a. Apa itu phpMyAdmin

**phpMyAdmin** adalah aplikasi berbasis web yang digunakan untuk mengelola **MySQL** melalui antarmuka grafis (GUI).
Aplikasi ini biasanya berjalan bersama **XAMPP, LAMPP, atau WAMP**.

### b. Fitur Utama phpMyAdmin

* Membuat, mengedit, dan menghapus database & tabel.
* Menjalankan perintah SQL tanpa terminal.
* Mengimpor & mengekspor database (format SQL, CSV, XML, dll).
* Mengatur hak akses user.
* Melihat relasi antar tabel (diagram ERD).
* Backup & restore database dengan mudah.

### c. Cara Mengakses phpMyAdmin

1. Jalankan **XAMPP Control Panel** â†’ Start **Apache** dan **MySQL**.
2. Buka browser â†’ ketik `http://localhost/phpmyadmin`.
3. Anda akan masuk ke dashboard phpMyAdmin.

## ğŸ§± **7. Hubungan MySQL dan PHP**

MySQL sering digunakan bersama **PHP** dalam pengembangan web:

* PHP berfungsi untuk memproses logika aplikasi.
* MySQL menyimpan data pengguna, produk, transaksi, dll.
* Keduanya berkomunikasi melalui fungsi atau ekstensi seperti:

  * `mysqli_connect()`, `mysqli_query()`
  * atau menggunakan **PDO (PHP Data Objects)**.

Contoh koneksi PHP ke MySQL:

```php
<?php
$koneksi = mysqli_connect("localhost", "root", "", "db_mahasiswa");

if (!$koneksi) {
    die("Koneksi gagal: " . mysqli_connect_error());
}
echo "Koneksi berhasil!";
?>
```
