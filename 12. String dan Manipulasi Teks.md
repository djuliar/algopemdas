Misalnya:

> "Halo, selamat pagi!"

Nah, di dalam dunia pemrograman, **teks seperti itu disebut â€œStringâ€**.

ðŸ‘‰ **String** adalah **sekumpulan karakter (huruf, angka, simbol, spasi, tanda baca, dll)** yang disimpan di dalam tanda kutip.

---

## ðŸ“˜ Contoh String

```python

```

ðŸ§  Catatan:

* Semua hal yang **diapit tanda kutip** (`" "` atau `' '`) dianggap **string**.
* Meskipun `angka = "12345"` terlihat seperti angka, sebenarnya itu **teks**, bukan angka matematis.

# ðŸ§± Karakter di Dalam String

String terdiri dari **karakter-karakter** yang bisa kita akses satu per satu menggunakan **indeks**.

ðŸ‘‰ Setiap karakter punya posisi (indeks) mulai dari **0**.

Contoh:

```python
teks = "Halo"
# Indeks:   0   1   2   3
# Huruf :   H   a   l   o

print(teks[0])   # H
print(teks[3])   # o
```

# âœï¸ Manipulasi Teks (String Manipulation)

â€œManipulasi teksâ€ artinya **mengolah isi string** â€” misalnya:

* Menggabungkan teks
* Mengubah huruf besar/kecil
* Menghitung panjang teks
* Mengambil sebagian teks
* Mencari atau mengganti kata

Kita bahas satu per satu ya ðŸ‘‡

---

## 1ï¸âƒ£ Menggabungkan String (Concatenation)

Kita bisa **menyambung** dua atau lebih string dengan tanda **`+`**.

```python
nama = "Andi"
salam = "Halo, " + nama + "!"
print(salam)
```

ðŸ§¾ Hasil:

```
Halo, Andi!
```

---

## 2ï¸âƒ£ Mengetahui Panjang String

Gunakan fungsi `len()` untuk mengetahui **berapa banyak karakter** dalam string.

```python
teks = "Pemrograman"
print(len(teks))
```

ðŸ§¾ Hasil:

```
11
```

Karena ada 11 huruf di kata â€œPemrogramanâ€.

---

## 3ï¸âƒ£ Mengubah Huruf Besar/Kecil

Kita bisa ubah teks jadi **huruf besar**, **huruf kecil**, atau **awalan kapital**.

```python
teks = "saya suka python"
print(teks.upper())   # semua huruf besar
print(teks.lower())   # semua huruf kecil
print(teks.title())   # huruf awal tiap kata besar
```

ðŸ§¾ Hasil:

```
SAYA SUKA PYTHON
saya suka python
Saya Suka Python
```

---

## 4ï¸âƒ£ Mengambil Sebagian Teks (Substring / Slicing)

Kita bisa ambil sebagian isi string dengan menggunakan tanda **`[ ]`** dan menentukan posisi awal & akhir.

```python
teks = "Pemrograman"
print(teks[0:3])   # dari indeks 0 sampai sebelum 3
print(teks[3:7])   # dari indeks 3 sampai sebelum 7
```

ðŸ§¾ Hasil:

```
Pem
prog
```

---

## 5ï¸âƒ£ Mencari dan Mengganti Kata

### Mencari kata dengan `in`

```python
kalimat = "Saya suka belajar Python"
print("Python" in kalimat)   # True (ada)
print("Java" in kalimat)     # False (tidak ada)
```

### Mengganti kata dengan `.replace()`

```python
teks = "Saya suka Python"
baru = teks.replace("Python", "Java")
print(baru)
```

ðŸ§¾ Hasil:

```
Saya suka Java
```

---

## 6ï¸âƒ£ Menghapus Spasi Berlebih

Gunakan `.strip()` untuk **menghapus spasi di awal dan akhir**.

```python
teks = "   Halo Dunia   "
print(teks.strip())
```

ðŸ§¾ Hasil:

```
Halo Dunia
```

---

## 7ï¸âƒ£ Menggabungkan dan Memecah Kalimat

### Menggabungkan daftar kata menjadi satu string

```python
kata = ["Saya", "suka", "Python"]
kalimat = " ".join(kata)
print(kalimat)
```

ðŸ§¾ Hasil:

```
Saya suka Python
```

### Memecah string menjadi daftar kata

```python
kalimat = "Belajar Python itu menyenangkan"
kata = kalimat.split()
print(kata)
```

ðŸ§¾ Hasil:

```
['Belajar', 'Python', 'itu', 'menyenangkan']
```

# ðŸ§  Hal yang Harus Diperhatikan

1. **String tidak bisa diubah langsung (immutable)** â€” kalau ingin mengubah, buat string baru.
2. Gunakan tanda kutip ganda `" "` atau tunggal `' '`, keduanya sama saja.
3. Gunakan **f-string** (fitur modern di Python) untuk gabung teks lebih rapi:

   ```python
   nama = "Andi"
   umur = 20
   print(f"Halo {nama}, umur kamu {umur} tahun.")
   ```
