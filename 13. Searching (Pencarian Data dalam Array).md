## ğŸ§  1. Linear Search (Pencarian Berurutan)

Linear search adalah **cara mencari data dengan memeriksa satu per satu** dari awal sampai akhir array.

### ğŸ’¡ Cara Kerja (Langkah Demi Langkah)

Misalnya kita punya array **sudah terurut**:

Langkah:

1. Lihat nilai **tengah** â†’ posisi ke-3 = 40.
2. 50 > 40 â†’ berarti cari di **kanan**.
3. Sekarang di kanan: [50, 60, 70]
4. Tengah = 60 â†’ 50 < 60 â†’ berarti cari di **kiri**.
5. Ketemu 50 âœ…

### ğŸ’» Contoh Program (Python)

```python
    tengah = (awal + akhir) // 2
    if data[tengah] == cari:
        ketemu = True
        print(f"Data {cari} ditemukan di indeks ke-{tengah}")
        break
    elif data[tengah] < cari:
        awal = tengah + 1
    else:
        akhir = tengah - 1

    print("Data tidak ditemukan")
```
